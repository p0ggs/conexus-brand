# Conexus Brand Asset Requirements

Based on the brand guidelines PDF analysis, here are all the required assets for the Conexus brand guidelines web application.

## Logo Assets

### Primary Logo Files
- **logo-light.png** - Logo for light backgrounds (PNG format, transparent background)
- **logo-light.svg** - Logo for light backgrounds (SVG format, vector)
- **logo-dark.png** - Logo for dark backgrounds (PNG format, transparent background)
- **logo-dark.svg** - Logo for dark backgrounds (SVG format, vector)
- **logo-color.png** - Full color logo with gradient (PNG format)
- **logo-color.svg** - Full color logo with gradient (SVG format)
- **logo-texture.png** - Logo with texture overlay (PNG format)
- **logo-texture.svg** - Logo with texture overlay (SVG format)

### Logo Specifications
- Logo consists of "CONEXUS" in Julius Sans One font
- "Wealth" in Crimson Pro serif font
- Extended "X" linking the two words
- Minimum spacing: 42px around logo in all directions
- Shapes in negative "X" spaces for flexible use

## Identity Shapes

### Shape Assets
- **identity-shapes.zip** - Collection of shapes extracted from the negative "X" spaces
  - Should include various vector shapes (SVG format)
  - Can be used as background elements
  - Flexible styling options for brand designs

## Texture & Gradient Assets

### Texture Files
- **texture-gradient.svg** - Primary gradient texture overlay (vector format)
- **texture-abstract.svg** - Abstract flowing vector shapes with gradient overlays (vector format)

### Gradient Specifications
- Primary gradient: #bbc1f7 → #a4d2f2
- Should be abstract, "flowing" vector shapes
- Use sparingly for brand logos and idents

## Typography Assets

### Font Files (if hosting locally)
- **Lato** - Primary typeface (Regular, Light, Semi-bold weights)
  - Used for: Body text, headings, buttons, forms
  - Available from Google Fonts
  
- **Crimson Pro** - Secondary typeface (Regular, Semi-bold weights)
  - Used for: "Wealth" in logo, key callouts
  - Available from Google Fonts
  - Use sparingly for typographical contrast
  
- **Julius Sans One** - Logo typeface (Regular weight only)
  - Used ONLY for: "CONEXUS" in main logo
  - Should NOT be used for text elements
  - Available from Google Fonts

## Color Assets

### Color Specifications (No files needed - colors are defined in CSS)
- **Conexus Dark**: #3b306a (RGB: 59, 48, 106, CMYK: 44, 55, 0, 58)
- **Ebony**: #151833 (RGB: 21, 24, 51, CMYK: 59, 53, 0, 80)
- **Conexus Light**: #dbebfa (RGB: 219, 235, 250, CMYK: 12, 6, 0, 2)
- **Primary Tones**: Various opacity levels (80%, 60%, 40%, 20%, 10%, 5%)
- **Gradient Colors**: 
  - Start: #bbc1f7 (RGB: 187, 193, 247, CMYK: 24, 22, 0, 3)
  - End: #a4d2f2 (RGB: 164, 210, 242, CMYK: 32, 13, 0, 5)

## Asset Specifications

### Logo Files
- **PNG**: Minimum 300 DPI for print, 72 DPI for web
- **SVG**: Vector format, optimized for web
- **Transparent backgrounds** where applicable
- **Multiple sizes** recommended (small, medium, large)

### Vector Files
- **SVG format** preferred for scalability
- **Optimized** for web use
- **Clean paths** without unnecessary nodes

### Package Files
- **ZIP format** for easy distribution
- **Organized folders** within ZIP files
- **Include README** in packages explaining contents

## Notes

1. All logo files should maintain the 42px minimum spacing requirement
2. Identity shapes should be provided in multiple formats (SVG, PNG)
3. Texture files should be vector-based for scalability
4. Ensure all assets are optimized for web use
5. Provide both RGB and CMYK color specifications where applicable
6. Logo variations should maintain consistent proportions
