# Assets Folder

Place your Conexus brand assets in this folder. See `ASSET_REQUIREMENTS.md` in the root directory for a complete list of required assets.

## Quick Reference

### Required Logo Files
- `logo-light.png` / `logo-light.svg` - Logo for light backgrounds
- `logo-dark.png` / `logo-dark.svg` - Logo for dark backgrounds
- `logo-color.png` / `logo-color.svg` - Full color logo with gradient
- `logo-texture.png` / `logo-texture.svg` - Logo with texture overlay

### Identity & Textures
- `identity-shapes.zip` - Shapes from negative "X" spaces
- `texture-gradient.svg` - Primary gradient texture
- `texture-abstract.svg` - Abstract flowing shapes

### Asset Packages
- `logo-pack.zip` - Complete logo set
- `icons.zip` - Brand icons (if applicable)
- `images.zip` - Stock photos (if applicable)
- `brand-guidelines.pdf` - Complete guidelines document

## File Organization

You can organize files directly in this folder, or create subfolders:
- `logos/` - All logo variations
- `shapes/` - Identity shapes
- `textures/` - Texture and gradient files
- `packages/` - ZIP packages

## Notes
- All files should match the exact names listed above
- PNG files: 300 DPI for print, 72 DPI for web
- SVG files: Optimized vector format
- Maintain 42px minimum spacing around logos
- See `ASSET_REQUIREMENTS.md` for detailed specifications
